Logo Stamp | Telegram Bot

A bot to make adding a single logo into many images easy. This bot is made to add (stamp) a predefined logo (or any image) to a given image (or images) right on telegram chat. It can be used through simple steps like this: the user send the logo image, set the position (coordinates) for the logo to be at and the size (scale) of the logo relative to the subject image size, and all done. The user then can send any image and get it back with the logo on it.
